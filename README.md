## What is this

Not a “complete Vim guide”.  
Not 500 commands.  
Not theory.

Just the stuff you need to **start using Vim without pain**.

---

## What’s inside

- modes (what’s going on at all)
- movement (how to stop using arrows)
- editing (delete / copy / paste / change)
- search & replace
- splits
- terminal
- a few things that save your нервная система

---

## Why

Because most Vim tutorials are:

- either overengineered  
- or useless in real life  

This one is neither.

---

## How to use

Open it.  
Keep it nearby.  
Use it while working.

That’s the whole idea.
